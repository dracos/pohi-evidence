1. Purpose of this Release

This document is the Release Notice for the IGL1 Sofware Baseline.

It contains the definition of the contents of the baseline and provides a checklist of media and
associated documentation to be used in the Build of the IGL1 Live system.

This software baseline reflects the software that was used during the UAT testing prior to 6/9/96.

This document does not cover the CAS sofiware that runs on VME.

2. Installation

The IGL1 Build Document, provided as part of this Release, covers the installation process for this
software.

3. Limitations of this Release

3.1 Outstanding problems found in UAT

At the time of publication of this document, further problems are being investigated which are likely to
require further updates to this software baseline. Such updates will be provided as ‘patches’ to this
baseline, and will be covered by separate Release notices.

Refer to PinICL for the latest status of problems found during UAT.
3.2 Examine STOP and EOI

The Help*Desk ‘Examine STOP and EOP utility provided with this release will only install on a
system that has the following visual basic libraries already installed:

VB40032.DLL
MSVCRT20.DLL.
OLEPRO32.DLL
MSVCRT40.DLL.

This problem will be rectified in a later release of this software.

3.3 Maestro Scripts

The Maestro scripts supplied with this release have still to be used in simulated operational
environment. This is planned for the re-run of the UAT commencing 09/09/96.
