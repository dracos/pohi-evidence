INTERVIEW RECORDING

1 PURPOSE
The aim of this policy is to inform Security Managers of the compulsory requirement to use the Quick
Reference Guide for interview recording procedures.

3. POLICY
It is mandatory for all Security Community employees to use the Quick Reference Guides when
conducting tape recorded interviews.
These guides can be found on the Corporate Security Database in the Library Section under:-
Interviewing - Quick Reference Guide.
