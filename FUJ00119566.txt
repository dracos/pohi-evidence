1. Purpose of this Document

This document is the Release Notice for updates to the IGL1 Software Baseline.

It contains the definition of the contents of the baseline and the changes made with each update. It
provides a checklist of media and associated documentation provided as part of each release

The revised baseline reflects the software that has been upgraded following UAT testing, in the period

between 11/9/96 and 17/9/96.

This document does not cover the CAS software that runs on VME.

2. Installation

The IGLI Software Update Build Document, provided as part of this Release, covers the installation

process for each update to the software baseeline.

3.2 Dependencies

These updates apply to the IGL1 Software Baseline which must be installed first. Details are covered in
the IGL1 Software Updates Build Document, reference TSC/PWY/018.

Each software update must be applied in turn as described in the Update Build Document.

3.3 Superseded Products

None, these are incremental updates to the software baseline described in IGL1 Build Document (Issue
2.0, 05/09/96) and IGL1 Software Updates (Issue 1.0, 11/09/96)

3.4 Limitations at time of Release

3.4.1 Outstanding problems found in UAT

At the time of publication of this document, further problems are being investigated which are likely to
require further updates to this software baseline, Such updates will be provided as ‘patches’ to this
baseline, and will be covered by separate Release notices.

Refer to PinICL for the latest status of problems found during UAT.
