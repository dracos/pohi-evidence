This release note describes the content of the software release, S90
1.2 Scope
This document defines the software changes and the platforms affected at release $90.
The S90 migration is described in the datacentre and counter migration strategies -
CS/STR/072 and CS/STR/073 respectively.

e CS/STR/072 — S90 & S92 Data Centre Migration Strategy
¢ CS/STR/073 — $90 Counter and PINPad Migration Strategy

Huthwaite Firewalls
